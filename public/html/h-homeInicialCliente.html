Raw Fitness

La evolución del modelo fitness ha provocado la necesidad de expansión industrial de este sector ya que cada vez más personas se ven involucradas en este movimiento de salud fitness, por eso el interés es tener un mejor estilo de vida estando desde casa.

Caracteristicas del producto

Calculadora de IMS

Una calculadora que hace la operacion matematica respetica para poder calcular el indice masa corporal de cada individuo.

Grafico de pesos

Existe un grafico de control de pesos para poder llevar el siguimiento del progreso personal de cada usuario.

Calendarizacion de actividades

Se calendariza cada actividad fisica, logro y cambio de peso para poder tener una nocion de progreso y poder eguir motivado con los cambios diarios.

Interfaz intuitivo

Un sistema amigable con el usuario para que se siente tan comodo como en casa y tenga una razon mas para seguiadelante.

Logros

Es bueno tener una perspecctiva de cambio aunque no se vea, por ende, se inluyo un sitema de logros adaptable a tu metas de cada dia y modificable si quiere hacer un cambio en el proceso.

Ejercicios

Tienes libertad de escoger como prefieres entrenar y con cual ejercicio te sientes mas comodo para tener un ambiente libre de presiones.
